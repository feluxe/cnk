
# cnk

### Description

"cnk" short for "chunk".

Split text (from stdin) into readable chunks that fit on one screen.


### Test

    git clone https://github.com/feluxe/cnk.git

    cd cnk

    cargo build

    python tests/gen.py 500 | ./target/debug/cnk
